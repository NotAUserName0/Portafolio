Red Social

Este proyecto es una red social pequeña, en la cual se puede publicar entradas, comentar, modificar usuario, eliminar entradas, comentarios y secciones. Se usó PHP, MySQL, HTML, CSS y JavaScript. El patrón de diseño usado en este provecto es MVC (Modelo-Vista-Controlador). Gracias a este proyecto logré entender el patrón de diseño MVC, manejo de datos obtenidos por una base de datos, así como Autenticación de usuarios.

Chat Realtime

Este es mi segundo proyecto ambicioso el cual es un chat pero en tiempo real, es decir, que al enviar un mensaje se ve reflejado al instante y sin recargar la pagina en ambos clientes. No solo eso, si no que tambien tiene un sistema el cual muestra a los usuarios(online,away,offline), asi como un pequeño sistema de 'amigos' el cual permite crear estan interacciones. Este proyecto esta en una solucion MEAN(MongoDB, Express, Angular, NodeJs), con un patron de diseño MVC(Modelo-Vista-Controlador). Lo que hace funcionar a esta aplicacion es que integro Socket.io para las actualizaciones en tiempo real y las comunicaciones en el chat, para separarlas de la conexion inicial En esta tambien concreto mis conocimientos en Autenticacion y Authorizacion con ayuda de un token JWT en el middleware de las peticiones.
